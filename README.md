# AI-Based Personal Finance Assistant Bot 🤖💰

An AI-powered personal finance assistant platform that helps users manage expenses, track budgets, and analyze spending habits. The system provides AI-based financial insights, expense categorization, and budgeting recommendations.

## 🚀 Key Project Features
- **MERN Architecture Structure**: Built using React.js, TypeScript, and Tailwind CSS.
- **Dynamic Analytics Dashboard**: Clean client-side state machine displaying real-time financial balances.
- **Visual Trend Breakdown**: Interactive percentage-based horizontal bar reports for category trends.
- **Search & Intelligent Filtering**: Real-time operational search bar with interactive multi-state filter tabs.
- **Gemini AI Simulation Engine**: Scans live data matrices to output tailored budget warning triggers and direct savings recommendations.
